NISRA X Hackathon

活動日期：2/27 (五) ~ 2/28 (六)


CODE is the only thing we use , HACK is the only thing we do.


活動預告

據說聖言樓的門都在晚上十點深鎖，外面的人再也進不來聖言的迴廊，但是為何一到了子夜時分，卻出現了人聲？

但這人聲並未帶來生機，而是令聖言樓就開始瀰漫著肅殺之氣！

瞬間！溫度降至冰河時期般冷峻，無數的怨靈從實驗室走出，拖著如殭屍般的步伐，兩眼無光、神情渙散、手指漫無目標地

在半空中比畫著，並且口中喃喃自語：


「為什麼有 crash？…crash…」


仔細一瞧，怨靈手上拿著一本書，為了一探那本書的究竟，擊敗了無數的怨靈。

發現這本書是以前那傳說中的 Programmer 寫的，據說上面寫著世界上最偉大的 code，滿懷期待的心打開那本書，結果發現上面寫著這麼一句話，

「最偉大的 code 由你 / 妳來創造，你 / 妳敢來挑戰嗎？」
