第二屆 NISRA Hackathon

活動日期：2/27 (五) ~ 2/28 (六)


CODE is the only thing we use , HACK is the only thing we do.

兩天一夜 連續 coding 28 小時

腦力激盪 盡情發揮自己的創造力

Hackathon = Hack + marathon

照字面翻譯便是不斷 Coding 的馬拉松。

在28小時內，像馬拉松一樣，近乎不眠不休的，構思與創造自己的程式碼

Hackathon 的精髓在於：很多人，在一段特定的時間內，相聚在一起，以他們想要的方式，去做他們想做的事情

最重要的是：整個編程的過程幾乎沒有任何限制或者方向。

你可能在半夜五點，還在思考問題solution，每分每秒都在與時間賽跑。
